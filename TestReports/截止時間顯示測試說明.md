## 技術細節

### 時間比較邏輯
```javascript
// 使用 dayjs 進行時間比較
const now = dayjs();
const isExpired = dayjs(group.deadline).isBefore(now);

// 檢查是否有可用店家
const hasAvailableStore = (event) => {
  return event.orderGroups.some(group => 
    dayjs(group.deadline).isAfter(dayjs())
  );
};
```

### 狀態判斷
- `isExpired = true` → 店家標籤灰色 + "(已截止)"
- `hasAvailableStore = true` → 顯示「進入點餐」按鈕
- `hasAvailableStore = false` → 顯示「已截單」禁用按鈕
